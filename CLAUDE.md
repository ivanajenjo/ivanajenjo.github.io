# Portfolio — Ivan Ajenjo

## Stack
- Astro v5 + Tailwind CSS v4
- Desplegado en GitHub Pages via GitHub Actions
- Repo: github.com/ivanajenjo/ivanajenjo.github.io
- URL actual: https://ivanajenjo.github.io
- Dominio futuro: ivanajenjo.dev (pendiente de configurar)

## Estructura
- `src/layouts/Layout.astro` — layout base
- `src/pages/index.astro` — página principal
- `src/components/` — Hero, SobreMi, Proyectos, Experiencia, Contacto, Nav
- `src/styles/global.css` — fuentes (Inter + Fira Code) y estilos base
- `.github/workflows/deploy.yml` — pipeline de despliegue

## Comandos
- `npm run dev` — servidor de desarrollo
- `npm run build` — build de producción

## Convenciones
- Idioma del código y contenido: inglés
- IDs de secciones en inglés: #about, #projects, #experience, #contact
- No hacer commits automáticamente, pedir confirmación
- No instalar dependencias sin preguntar
